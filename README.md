<h4>개인 프로젝트 소개</h4>
<hr color='#D5D5D5'>
사용자들에게 편의성을 주기 위하여 웹을 통해 물품을 구매할 수 있는 웹 사이트를 제작하였습니다.

<h4>구현 기능</h4>
<hr color='#D5D5D5'>
회원가입, 로그인, 물품 리스트, 카테고리, 
<b>장바구니</b>, <b>구매</b>, 구매 내역, <b>게시판(문의)</b>, 
비로그인 시 구매 및 장바구니 제한
